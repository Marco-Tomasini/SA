Regras de Negócio: 

| RN001  | Todos os trabalhadores precisam de um cadastro com senha. |
| :---- | :---- |
| RN002  | O sistema deve notificar todos os trabalhadores sobre qualquer aviso. |
| RN003  | Os trabalhadores podem acionar a equipe de manutenção. |
| RN004  | O administrador pode realizar uma parada de emergência de um trem através do botão de mensagem de emergência |
| RN005  | O sistema deve permitir redefinição de senha via e-mail. |
| RN006  | Os trabalhadores podem alterar o status das rotas. |
| RN007  | Toda viagem tem origem e destino definidos. |
